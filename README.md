## Section Accuracy Overlay
Tosu Overlay  
Currently supports taiko and mania.  
[Supported Dan List](/DanList.txt)  

If there’s a Dan map that isn’t supported yet, please add it to `data.json`.

### Installation
1. Install [Tosu](https://tosu.app)
2. Download `section-accuracy-overlay.zip` from the [Releases](https://github.com/shaaaaaQ/section-accuracy-overlay/releases) page.
3. Extract the ZIP file.
4. Place the extracted folder into Tosu’s `static` directory.
